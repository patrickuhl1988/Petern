# Heute Petern? – Dein Entscheidungshelfer

Eine kleine humorvolle Web-App: Du gibst eine Aktivität ein, klickst – und mit **90 % Wahrscheinlichkeit** bekommst du eine lockere Ausrede, warum du lieber auf der Couch bleibst. Mit **10 %** darfst du doch raus.

## Features

- **Entscheidung:** Aktivität eingeben (z.B. Feiern, Sport, Kino) → Button → Ausrede oder „Darfst du“.
- **Schnellauswahl:** Chips für Feiern, Sport, Kino, Treffen, Früh aufstehen, Einkaufen.
- **Ausreden:** Locker, vage, relatable (durch, morgen, keine Energie, was dazwischengekommen, etc.).
- **Petern-Level:** Zufällige Stufe wie „Sofa-Pilot“ oder „Legende der Ausrede“.
- **Ausrede kopieren:** Ein Klick kopiert die Ausrede in die Zwischenablage.
- **Letzte Entscheidungen:** Kurze Historie (localStorage).
- **Petern des Tages:** Eine Tages-Ausrede, auf Wunsch neu würfeln.

## Technik

- Nur HTML, CSS und JavaScript – keine Abhängigkeiten.
- Lokaler Speicher für Verlauf und Tages-Ausrede.

## Lokal starten

Dateien lokal öffnen oder einen einfachen Server starten:

```bash
# z.B. mit Python
python -m http.server 8000
# Dann http://localhost:8000 im Browser öffnen
```

Oder `index.html` direkt im Browser öffnen.

## Lizenz

Spielerei – nutzen wie du magst.
